Какие акции бывают на Ozon
Чтобы увеличить продажи и получить больше заказов, вы можете продвигать свои товары через акции. Есть два варианта:

добавить товары в одну из акций, которые мы проводим для покупателей;
запустить собственную акцию и самостоятельно определить сроки, товары, условия и т.д.
Если товар участвует в акциях, он исчезнет из всех акций после автоприменения рыночной цены — если новая цена ниже цены по акции. Также товар может пропасть из акции, если вы меняете цену вручную. Подробнее об управлении ценами

Акции от Ozon
Мы регулярно проводим акции для покупателей. Сроки и условия для них определяем сами, а вам остаётся только добавить туда свои товары в разделе Продвижение → Акции. Они будут показываться на страницах акций вместе с продукцией Ozon и других партнёров.

Участие в таких акциях бесплатное, но скидки будут за ваш счёт. Виды акций, к которым можно присоединиться через личный кабинет:

Скидка. Она устанавливается в процентах и зависит от Цены до скидки или Текущей цены, которая действовала последние 30 дней. Подробнее о прямых скидках
Скидка на сток. Механика похожа на предыдущую, но может рассчитываться от Цены по акции. За несколько дней до старта доступ в такую акцию закрывается, а с некоторыми продавцами мы даже заранее можем согласовывать список и количество товаров.
Самыми массовыми являются глобальные акции «Маркетплейс промо». Их мы проводим несколько раз в год и дополнительно продвигаем:

на сайте Ozon;
в рассылках на электронную почту покупателям Ozon;
в push-уведомлениях пользователям мобильного приложения Ozon;
в историях Ozon в Instagram.
Иногда мы сами выбираем товары для участия в наших акциях. Тогда скидка предоставляется за наш счёт — вы получите полную стоимость за вычетом комиссии, поэтому в личном кабинете и отчётах везде будет указана ваша цена.

Мои акции
Вы можете запускать собственные акции на Ozon и самостоятельно их продвигать. Для них сможете сами определить сроки, ассортимент и размер скидок.

Виды акций, которые можно спланировать и настроить самостоятельно:

С обычной скидкой. В такую акцию вы можете добавить любые товары и установить на них любую скидку. Если ваш товар участвует в нескольких акциях одновременно, то он будет продаваться по самой низкой из цен по акции.
Купон. Это промокод, с помощью которого покупатель может получить скидку на покупку. Подробнее о купонах

Как устроены аудитории для акций
Некоторые акции Ozon предназначены для определённых клиентов — например, только для тех, кто заказывал товары для праздников. В таком случае рядом с названием будет тег Аудитория, а в описании — категорий покупателей, которым доступна акция.



Где посмотреть, в каких акциях могут участвовать товары
Для этого откройте раздел Продвижение → Акции и выберите вкладку Доступные — появится список открытых для вас акций и сроки их проведения.

Почему не получается добавить товар в акцию
Возможные причины:

вы создали товар меньше недели назад;
позиции нет в наличии или остатки на складе равны 0;
товар не подходит под условия акции Ozon. Система автоматически определяет список — а значит, мы не сможем включить в акцию другие позиции даже по запросу в поддержку.
Почему товар исчез из акции
Возможные причины:

было автоприменение рыночной цен, и новая цена ниже цены по акции;
вы изменили цену вручную.

Как узнать, в каких акциях участвуют товары
Откройте раздел Продвижение → Акции и выберите вкладку Участвую.

Кликните на нужную акцию — откроется страница со сроками проведения и списком товаров.

Если позиции из списка окажутся ещё в какой-то акции, появится столбец Есть в других акциях, а в нём:

синий круг — если товар одновременно участвует в акциях одного уровня;
жёлтый треугольник — если товар одновременно участвует в акциях разного уровня.


Нажмите на значок, чтобы посмотреть подробную информацию. Откроется окно Пересечение с акцией, где в блоках:

Акции за указанный период — акции уровня 1;
Акции на данный период — акции Уровня 2;
Как будет изменяться цена — данные о стоимости товара с учётом акций. В таблице можно посмотреть:
период и длительность — сколько времени товар будет продаваться по этой цене. Например, 7-9 февраля, 3 дня.
цена — стоимость с учётом всех скидок по акциям, доступных всем покупателям на Ozon.
минимальная цена — минимальная возможная цена, если к цене по акции будет применена скидка по акции.


Что произойдёт, если товар в нескольких акциях
Покупатель увидит:

на странице карточки товара — стикеры со всеми акциями;
в поиске и каталоге — стикер с самой выгодной акцией.
Итоговую стоимость товара система рассчитает автоматически в зависимости от того, к какому уровню относится каждая акция:

уровень 1 — типа «Скидка»;
уровень 2 — типа «Купоны».
Если товар участвует в акциях одного уровня
В нескольких акциях типа «Скидка» — используем самую низкую из цен по акции. Когда одна из акций закончится, товар будет продаваться по цене из текущей акции.
Пример. Первая акция проходит с 1 по 15 апреля, цена товара — 500 рублей. Вторая акция идёт с 10 по 20 апреля, цена — 600 рублей. Тогда с 1 по 15 апреля товар будет продаваться за 500 рублей, а с 16 по 20 апреля цена поднимется до цены по второй акции — 600 рублей.

В «Комплекте» и любой из акций уровня — товар будет продаваться по скидке, установленной на комплект.
При любых комбинациях акций уровня 2, кроме типа «Комплект» — используем самую низкую из цен по акции.
Если товар в акциях разного уровня
Тогда к товару применятся обе скидки. Стоимость товара по второй акции будет рассчитана из цены со скидкой, по которой товар продаётся в первой акции.

Пример.

Исходная цена товара — 1000 рублей. Товар участвует в акции типа «Скидка», цена товара по этой акции — 900 рублей. Потом товар добавляют в акцию «Купоны» с установленной скидкой 10%. Стоимость товара по этой акции рассчитывается из 900 рублей: 900 - 900 × 10% = 810. Итоговая стоимость товара составит 810 рублей.

Как узнать, по какой акции куплен товар
Узнать акцию, по которой куплен товар, можно в разделе Заказы. Для этого откройте страницу с нужным заказом и наведите указатель на цену товара.



Как удалить товар из акции
Откройте раздел Продвижение → Акции и выберите вкладку Участвую.
Кликните на нужную акцию — откроется страница со сроками проведения и списком товаров.
Выберите нужные товары и нажмите Удалить.


Как посмотреть отчёт по завершённой акции
Чтобы посмотреть статистику по завершенной акции:

Перейдите в раздел Продвижение → Акции → Завершенные.
На карточке с акцией нажмите ⋮ → Скачать статистику.
На ваш компьютер будет загружен XLS-файл со статистикой, в котором будет 3 листа: описание, результаты акции, данные для сравнения.

Описание
Содержит общую информацию по акции:

Название акции;
Тип акции;
Аудитория — категория покупателей Ozon, для которой предназначена акция;
Период акции;
Условия участия в акции.
Результаты акции
Содержит результаты участия в акции по каждому товару:

Цены:

Последняя цена продажи до начала акции;
Цена по акции;
Продолжительность участия товара в акции по такой цене;
Период участия товара в акции по такой цене.
Продажи:

Продано, шт.;
Продано, руб.
Если товар участвовал в нескольких акциях одновременно, то он продавался по самой низкой из цен по акции. Если товар был продан по цене из другой акции, в полях Продано, шт. и Продано, руб. будет указан 0 (даже если товар купили со страницы текущей акции).

Результаты участия в акции по одному и тому же товару показаны в нескольких строках, если:
в период проведения акции цена по акции (скидка) на товар менялась;

товар был исключен из акции, а позже добавлен снова.

Данные для сравнения
Содержит:

сравнение результатов участия в акции с показателями за неделю до акции — средней ценой и количеством проданного товара;
список более ранних акций, в которых участвовал товар.
Как посмотреть аналитику по завершённым акциям
Для этого откройте раздел Аналитика → Акции — покажем ваши результаты на графиках.
